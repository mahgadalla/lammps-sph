* Intorduction
Poiseuille flow with parameters from Morris [1] and unisg reverse
Poiseuille flow [2].

| eta | 1e-6 | m^2 s^-1 |
| L   | 1e-3 | m        |
| rho |  1e3 | kg m^-3  |
| F   | 1e-4 | m s^-2   |

* Setup

Run run.sh. After simulation finished use gnuplot file poiseuille.gp
to see the plots.

* Footnotes

Incompressible Flows Using SPH. Journal of Computational Physics 136,
