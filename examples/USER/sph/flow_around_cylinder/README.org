Flow around cylinder from Morris et al. [1]. 

| eta |    1e-6 | m^2 s^-1 |
| L   |    1e-1 | m        |
| R   |    2e-2 | m        |
| rho |     1e3 | kg m^-3  |
| F   |  1.5e-7 | m s^-2   |
| c   | 1.25e-4 | m s^-1   |

* Setup

Run run.sh. After simulation finished use gnuplot file flow.gp
to see the plots.

To load particle into vmd run:
vmd -e post.tcl -dcd data/data.dcd
